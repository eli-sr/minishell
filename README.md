# Minishell

This is a simple shell programmed in C that accepts one pipe and bidirectional files redirection.

## Installation
1. Clone repository
```
git clone https://github.com/eli-sr/minishell.git
```

2. Go to directory
```
cd minishell
```

3. Compile
```
make
```

4. Try the shell

```
./minishell
```
